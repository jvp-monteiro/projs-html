Comandos Git - Explicação Técnica e Simples

git init

Inicializa um novo repositório Git no diretório atual.

git add .

Adiciona todos os arquivos modificados ou novos ao "staging area" (área de preparação para commit).

git add arquivo.txt

Adiciona o arquivo arquivo.txt ao staging area.

git commit -am "mensagem"

Registra as mudanças com uma mensagem. O -a inclui arquivos já monitorados, e o -m adiciona a mensagem.

git checkout -b feat/novo

Cria uma nova branch chamada feat/novo e muda para ela.

git checkout feat/novo

Alterna para a branch feat/novo.

git remote add origin URL_GITHUB

Adiciona o repositório remoto com apelido origin.

git push -u origin feat/novo

Envia a branch atual para o GitHub e define origin como o remoto padrão para essa branch.

git pull

Baixa e incorpora mudanças do repositório remoto para o local.

git log

Exibe o histórico de commits da branch atual.

git branch

Lista todas as branches e indica qual está atualmente selecionada.

git branch -M novo-nome

Renomeia a branch atual para novo-nome.

git clone URL_GITHUB

Clona um repositório remoto (no GitHub) para o seu computador.
