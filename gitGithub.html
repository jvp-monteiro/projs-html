Entenda o que é Git e GitHub

Ferramentas essenciais para qualquer desenvolvedor

📁 O que é Git?

Git é um sistema de controle de versão. Ele permite que você acompanhe o histórico de mudanças feitas no código-fonte de um projeto. É como um "salvamento inteligente" do seu progresso, permitindo que você volte atrás, crie versões paralelas (branches), e trabalhe em equipe sem sobrescrever o trabalho dos outros.

  • Armazena versões dos arquivos
  • Ajuda a organizar o desenvolvimento
  • Funciona localmente (sem internet)

🌐 O que é GitHub?

GitHub é uma plataforma online que hospeda repositórios Git. Ele funciona como uma "nuvem" para seus projetos, onde você pode colaborar com outras pessoas, revisar código, abrir issues (problemas/sugestões), e fazer deploy de aplicações.

  • Hospeda projetos com Git
  • Facilita colaboração entre desenvolvedores
  • Permite controle de acesso e revisão de código

🔗 Git vs GitHub

Git é a ferramenta que você usa no seu computador. GitHub é o serviço online onde você pode guardar e compartilhar os projetos versionados com Git.

Você pode usar Git sem GitHub, mas o GitHub precisa do Git para funcionar.
